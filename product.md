# Product Context: Muldan Portfolio

## Overview
A dynamic, developer-focused portfolio website for "Muldan", an Independent Automation Consultant. The site acts as a digital business card and interactive showcase of data extraction and automation capabilities.

## Core Identity
- **Role:** Automation Consultant / Data Ops / Sys Admin
- **Specialty:** Building scrapers, automated systems, bypassing Cloudflare, and custom API glue.
- **Vibe:** Technical, terminal-inspired, hacker-chic but professional (Data Ops).

## Key Features
1. **Interactive Case Studies:**
   - Instead of static images, case studies feature interactive "Simulations" displaying a mock terminal console alongside a live-updating data table.
   - Users can switch between projects (e.g., Lead Scraper, Price Tracker, Webhook Integrations) and watch the automation run in real-time.
2. **Terminal Aesthetics:**
   - Monospace typography.
   - Grid-based layouts and borders.
   - Status indicators (`[SYS.ADMIN]`, `AVAILABLE`).
3. **Responsive Design:**
   - Mobile-friendly grid system that adapts the terminal simulation to smaller screens.
